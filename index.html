SOBRE MIM

Sempre fui movido pela curiosidade de entender como as coisas funcionam. Cursando Análise e Desenvolvimento de Sistemas na FATEC, encontrei no Frontend a combinação perfeita para unir a lógica da programação com a criatividade do design.

Para mim, o código é o meio para criar experiências incríveis. Hoje, transformo layouts e ideias em interfaces reais e interativas, utilizando HTML, CSS e JavaScript como base sólida, enquanto avanço na construção de aplicações modernas com React.
